# Project Context Kit Template

Copy this kit into a new project root and replace bracketed text. Keep detailed
designs, research, and task evidence in their existing folders; these files are
the compact entry layer for people and agents.

Required files:

- `AGENTS.md`
- `PLAN.md`
- `PROGRESS.md`
- `DECISIONS.md`
- `TASKS.md`
- `MEMORY.md`
- `CHANGELOG.md`
- `ARCHITECTURE.md`

Read `docs/operations/universal-work-context-workflow.md` for lifecycle and
update rules. For coordinated software projects, point `TASKS.md` to
`coordination/task-board/` rather than duplicating task status.
